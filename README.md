# Compteur de Clics

Un projet simple pour aprendre JavaScript.

## Description

Ce projet est un compteur interactif qui permet d'incrementer, décrémenter et réinitialiser un compteur.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript ES5

## Installation

1. Cloner le repo
2. Ouvrir index.html dans votre navigateur

## Fonctionnalités

- Bouton + pour augmenter le compteur
- Bouton - pour diminuer le compteur
- Bouton reset pour remettre à zéro

## Contribuer

Pour contribuer à ce projet :

1. Cliquer sur le bouton **Fork** en haut à droite du repo GitHub
2. Cloner votre fork sur votre machine :
   ```bash
   git clone https://github.com/VOTRE-USERNAME/r-vision-git.git
   ```
3. Créer une branche pour vos modifications :
   ```bash
   git checkout -b ma-branche
   ```
4. Faire vos modifications et les commiter :
   ```bash
   git add .
   git commit -m "Description de mes modifications"
   ```
5. Pousser votre branche vers votre fork :
   ```bash
   git push origin ma-branche
   ```
6. Ouvrir une Pull Request depuis GitHub vers le repo original
